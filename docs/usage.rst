Install the package
-------------------

We recommend using a virtual environment via ``virtualenv`` or ``conda``, and use ``pip`` to install ``iscan-dag``.
To use iSCAN, first install it using pip:

.. code-block:: console

   (venv) $ pip install --upgrade iscan-dag
